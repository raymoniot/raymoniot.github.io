从设备列表中的设备可以进入设备详情页

3.1报警

3.2解绑

参考采集器解绑，子设备的解绑将会从采集器下删除该子设备，清空子设备的数据。

3.3新建工单

参考采集器新建工单

3.4参数设置

通过绿色按钮进行测点参数设置下发，或者点击批量设置下发，详细请参考远程交互链接

3.5别名编辑

参考采集器别名编辑

3.6固件升级

参考采集器固件升级

3.7操作日志

参考采集器操作日志

3.8历史数据图表

历史数据支持日、周、月、年、总级别的数据展示和时间范围选择，其中不同的时间维度展示的参数不同

  • 日：通过参数选择用户可以自定义参数展示
  • 周：通过参数选择用户可以自定义参数展示
  • 月：展示发电量
  • 年：展示发电量
  • 总：展示发电量

3.8.1参数选择

用户可以选择想要展示的参数，点击确定查看

用户也可以自定义模板，每个模板可以保存自定的参数

3.8.2导出报表

将图表数据导出为Excel表格
